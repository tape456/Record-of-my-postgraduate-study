案例:

    要求:两个张量的维度保持一致,对应元素直接做 相应的操作
    API:
        t1 * t2
        t1.mul(t2)          #multiply:乘法

    要求:两个张量,第一个张量的列数,等于第二个张量的行数 (A列 = B行)

        t1 @ t2

import torch

    print(f"t1:{t1}")

    # 2.定义张量,3行2列

    # 3.演示矩阵乘法操作,对应元素直接做 相应的操作
    # t3=t1 @ t2
    # t3=t1.matmul(t2)       # 效果同上 matrix multiply(矩阵乘法)

    # 4.打印结果
    # print(f"t3:{t3}")

    # 5.演示dot()函数
    # t3=t1.dot(t2)           # 报错，dot()只针对于一维张量有效
    t4=torch.tensor([1,2,3])
    t5=torch.tensor([4,5,6])
    t6=t4.dot(t5)
    print(f"t6:{t6}")

# 3.测试
if __name__ == '__main__':
    # dm01()

#### 梯度下降法求最优解

梯度形式是不变的,梯度的值是变的(2w)



![image-20260317205405512](深度学习-img/image-20260317205405512.png)



![image-20260317213827587](深度学习-img/image-20260317213827587.png)

```py
"""
演示自动微分模块,循环实现 计算梯度,更新参数

需求:
    求 y = x**2 + 20 的极小值点 并打印y是最小值时 w的值(梯度)

解题步骤:
# 1. 定义点 w=10 requires_grad=True  dtpe=torc
# 2. 定义函数 loss = w**2 + 20
# 3. 利用梯度下降法 循环迭代1000 求最优解
# 3.1 正向计算(前向传播)
# 3.2 梯度清零 w.grad.zero_()
# 3.3 反向传播
# 3.4 梯度更新 w.data = w.data - 0.01 * w.grad

回顾
    W新=W旧-学习率 * 梯度
"""
import torch

# 1. 定义点 w=10 requires_grad=True  dtpe=torch.float32
# 参1:初始的权重 w(旧),参2:自动微分 (求导),参数3:数据类型 浮点型
w=torch.tensor(10,requires_grad= True,dtype=torch.float32)

# 2. 定义函数 loss = w**2 + 20
loss=w**2 + 20          # 求导:loss'=2w


# 3. 利用梯度下降法 循环迭代1000 求最优解
print(f"开始 权重初始值:{w},(0.01*w.grad):无,loss:{loss}")

# 迭代100次,求最优解   [其实执行了101次]
for i in range(1,101):
    # 3.1 正向计算(前向传播)
    loss=w**2 + 20

    # 3.2 梯度清零 w.grad.zero_()   默认梯度会累加
    # 至此(第一次的时候),还没有计算梯度,所以 w.grad=None,要做非空判断
    if w.grad is not None:
        w.grad.zero_()

    # 3.3 反向传播
    loss.sum().backward()

    # 3.4 梯度更新 w.data = w.data - 0.01 * w.grad        因为梯度是2w，这里的w.grad就是把 w 带进去了得到梯度值
    # print(f"梯度值为:{w.grad}")
    w.data = w.data - 0.01 * w.grad

    # 3.5 打印本次 梯度更新后 权重参数结果
    print(f"第{i}次 权重初始值:{w},(0.01*w.grad):{0.01*w.grad:.5f},loss:{loss:.5f}")


# 4.打印最终结果
print(f"最终结果   权重:{w},梯度:{w.grad:.5f},loss:{loss:.5f}")
```

### 损失函数求导得到梯度，损失函数具体到底是对谁求导？？

损失函数求导（计算梯度）的核心目标是：**对模型中所有可学习的参数（权重 / 偏置）求导**，最终目的是通过梯度下降更新这些参数，让模型预测更接近真实值。

**模型参数**：神经网络中需要学习的变量，比如全连接层的权重 `W`、偏置 `b`，卷积层的卷积核等（PyTorch 中这些参数会被标记为 `requires_grad=True`）。

**求导本质**：损失函数是模型参数的 “函数”（因为预测值由参数计算而来），求导就是计算 “损失对每个参数的变化率”—— 参数变一点点，损失会变多少。

### 梯度形式是不变的，梯度值是变的

```py
# 2. 定义函数 loss = w**2 + 20
loss=w**2 + 20          # 求导:loss'=2w
```

传播过程中：

梯度（导数结果式子）形式永远是 2w

但是梯度值是变的（权重 w 一直在更新）
