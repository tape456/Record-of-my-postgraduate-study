## 不记得终端输入什么指令了？

在终端里跑这几个命令，不用 tmux：

```bash
# 1. 看所有正在运行的 Python 进程及完整命令(常用)
ps aux | grep python | grep train_data_collection

# 2. 更精确，看完整命令行（不会被截断）
ps -eo pid,args | grep train_data_collection

# 3. 看 tmux 里某个 session 的历史（如果 buffer 够大）
tmux capture-pane -t 你的session名 -p -S -5000 | grep "python train"
tmux capture-pane -t meta -p -S -5000 | grep "python train"
```
