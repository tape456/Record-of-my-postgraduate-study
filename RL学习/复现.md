## 不记得终端输入什么指令了？

在终端里跑这几个命令，不用 tmux：

```bash
# 1. 看所有正在运行的 Python 进程及完整命令(常用)
ps aux | grep python | grep train_data_collection

# 2. 更精确，看完整命令行（不会被截断）
ps -eo pid,args | grep train_data_collection

# 3. 看 tmux 里某个 session 的历史（如果 buffer 够大）
tmux capture-pane -t 你的session名 -p -S -5000 | grep "python train"
tmux capture-pane -t meta -p -S -5000 | grep "python train"
```

## tmux显示每个窗口的具体运行命令

方案 3：自定义快捷键，一步查看（永久生效）

编辑 tmux 配置：

```
vim ~/.tmux.conf
```

在 `~/.tmux.conf` 添加：

```
set -g pane-border-status bottom
# 窗格下方显示完整运行命令
set -g pane-border-format "#P: #(ps -o args= -t #{pane_tty} | cut -c1-80)"
```

`source ~/.tmux.conf` 后，每个分屏底部会一直展示当前训练脚本，不用每次输命令查询。

备选：不进 tmux，外部终端重载

在 ssh 终端（不在 tmux 内）执行：

```
tmux source ~/.tmux.conf
```
