## 为什么 `range(3)` 是 `0,1,2`？

结合语法规则拆解：

- `range(3)` 属于第一种形式（仅传 `stop=3`），因此默认 `start=0`，`step=1`
- 按「左闭右开」规则，序列需满足：`start ≤ x < stop`（x 是整数，步长 1）
- 代入数值：`0 ≤ x < 3`，满足条件的整数只有 `0, 1, 2`

> 用 **%d** 和 **%f** 有专门的好处：可以做精度的控制  (数字专有类型)

%d 的英文全称是decimal（十进制），而不是integer（整数）

```py
"""
演示使用%占位的方式对字符串进行格式化（拼接）
"""

name="周杰伦"
age=11
height=172.55

#传统拼接
message="我是"+name+"，今年"+str(age)+"岁，身高："+str(height)
print(message)

# %占位符
message2="我是%s," % name         #将name的值填入%s的位置
print(message2)

# %s占位，但是提供数据类型，则会自动转为字符串
message3="我是%s，今年%s岁，身高：%s" % (name,age,height)
print(message3)

# %s 占位是字符串
# %d 占位是数字
# %f 占位是浮点数
message4="我是%s，今年%d岁，身高：%f" % (name,age,height)
message4="我是%s，今年%d岁，身高：%.1f" % (name,age,height)      #精度控制会四舍五入
print(message4)
```

```py
print("{1}:计算机{0}的CPU占用率为{2}%".format("2026-3-11",'C',10))

# 格式控制符格式：<填充> <对齐方式> <宽度> <,> <.精度> <类型>
s='Python'
print("{0:30}".format(s))       #填充和对齐方式只有在宽度确定的情况下才有意义，所以这里的30是宽度
print("{0:>30}".format(s))       #右对齐
print("{0:^30}".format(s))       #居中对齐
print("{0:<30}".format(s))       #默认左对齐
print("{0:-^30}".format(s))       #居中对齐，没有字符的地方用-填充
print("{0:-^30,}".format(12346978))       #居中对齐，没有字符的地方用-填充,把数据类型的整数部分按三位去划分
print("{0:-^30}".format(12346978))       #居中对齐，没有字符的地方用-填充
print("{0:-^30,.2f}".format(12456.9658))       #居中对齐，没有字符的地方用-填充,把数据类型的整数部分按三位去划分,保留2位小数
print("{0:-^30.2}".format(s))       #精度作用在字符串表示保留几位
#类型：用来指定我们格式化的数据类型是什么
print("{0:b},{0:c},{0:d},{0:o},{0:x},{0:X}".format(123))
# b二进制数 c输出整数对应的Unicode字符 d输出整数的十进制方式 o八进制 x小写的十六进制数 X大写的十六进制数
print("{0:e}--{0:E}--{0:f}--{0:%}".format(3.14))
# e输出对应小写字母e的指数形式 E输出对应大写字母E的指数形式 f输出浮点数的标准浮点形式 %输出浮点数的百分形式

# format()方法在处理 s格式符时，会检查传入的对象。
# 虽然浮点数在打印时能显示为文本，但 s要求对象具有 __str__方法且能安全地被视为字符串。
# 对于非字符串对象，Python 3 的 format()会严格检查类型。
print("{0:s}".format(str(3.14)))     #输出字符串形式
```

内置变量的作用：

```py
# 如果本文件被直接执行，则内置变量 __name__会被赋值为："__main__"
# 如果本文件被import或from导入，则内置变量__name__会被赋值为：文件名称
# __name__是Python内置变量，任何代码文件都有
# 我们可以通过读取这个变量的值，从而确定这个代码文件是：被人导入了 还是被作为程序执行了

# 如果你觉得我们这个文件有可能被人导入，那就把所有的函数调用都放到这个里面去

if __name__=="__main__":        # 也就是说如果这个文件被作为程序执行了，那__name__就被赋值为__main__,下面的代码就能执行
    wangwang()
```
