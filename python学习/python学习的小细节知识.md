## 为什么 `range(3)` 是 `0,1,2`？

结合语法规则拆解：

- `range(3)` 属于第一种形式（仅传 `stop=3`），因此默认 `start=0`，`step=1`
- 按「左闭右开」规则，序列需满足：`start ≤ x < stop`（x 是整数，步长 1）
- 代入数值：`0 ≤ x < 3`，满足条件的整数只有 `0, 1, 2`
